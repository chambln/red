Author’s name

10th July 2019

The Makefile uses the shell command sort -r to process posts in reverse alphabetical order, so if you want your posts to appear in reverse chronological order in the archive list, you must name your files with the date at the front, in a format such as YYYY-MM-DD-*.md.

Pandoc, however, will print the date that you provide in the YAML metadata at the top of each markdown post. It also prints the title, author(s), and some other metadata like description which appear in the archive list.
